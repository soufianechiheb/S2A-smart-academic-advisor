## Goal
Provide personalized academic guidance using a Knowledge Graph and Semantic Web technologies.

## Target user
University student seeking guidance for:
- course planning,
- research exploration,
- career orientation.

## Core features (mandatory)
1. Student profile definition (interests, completed courses).
2. Course recommendation based on prerequisites and interests.
3. Discovery of relevant research papers linked to courses.
4. Mapping skills acquired to potential career paths.
5. Semantic queries via SPARQL.

## Out of scope (not implemented)
- Real-time university systems integration.
- Machine learning-based recommendations.
- Authentication / user accounts.

## Demo scenarios
1. "Which course should I take next?"
2. "Which research papers relate to my courses?"
3. "Which careers match my current skills?"
